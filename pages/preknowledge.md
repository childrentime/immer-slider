## 可变数据

<br/>

```ts
const a = 1;
const b = a;
b = 2;

const x = {
  a: 1,
};
const y = x;
x.a = 2;
```

</div>
<div>

## 不可变数据
<br/>

```ts
const x = {
  a: 1,
};
const y = { ...x };
y.a = 2;

const x = { a: 1, b: { c: 2 } };
const y = Object.assign({}, x, {
  b: Object.assign({}, x.b),
});

y.b.c = 22;
```





</div>
</div>

<br/>
<br/>
不可变对象的定义: 不可变对象则是创建之后，不能再修改状态，对其任何删改操作，都应返回一个新的对象
